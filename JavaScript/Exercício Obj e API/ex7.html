API Dota 2

API de Dota 2 criada para que fosse possível buscar informações sobre player base, PRO players, informações de partidas, heros mais pickados etc. Num resumo, esta API fornece toda a informação do jogo para que sejam seja criado estatísticas precisas e análises cada vez mais profundas sobre o jogo. Possui uma documentação bem trabalhada com exemplos de chamadas para todas a informações necessários para conseguirmos trabalhar os dados. Resolvi pegar os dados sobre última partida do TI 2019 para pegar os heroes pickados, quantidade e informações sobre as teamfigths.
